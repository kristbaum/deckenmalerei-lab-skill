# Deckenmalerei-Lab Skill

Skill für die Nutzung des [Deckenmalerei-Lab](https://badwcbd-lab.srv.mwn.de/).

## Installation

### Claude.ai

Den Skill als **Projekt-Anweisung** in Claude.ai einrichten:

1. Auf [claude.ai](https://claude.ai) ein neues **Projekt** erstellen
2. In den Projekteinstellungen unter „Anweisungen" den Inhalt der Skill-Datei einfügen
3. Ab jetzt steht der Skill in allen Chats innerhalb dieses Projekts zur Verfügung

### ChatGPT

Den Skill als **Custom GPT** einrichten:

1. Öffne den GPT Builder unter „Explore GPTs" → „Create"
2. Kopiere den Inhalt der Skill-Datei in das Feld „Instructions"
3. Speichere den Custom GPT – er steht danach in der GPT-Übersicht zur Verfügung
